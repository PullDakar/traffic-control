## Для установки необходимо
- Установить Linux/Windows ОС
- Установить на сервер Open JDK 11
- Разархивировать файл jat.zip 
## Для запуска приложения необходимо:
- Запустить консоль
- Перейти в каталог, в который разархивирован дистрибутив
- Запустить приложение командой
```
java –jar traffic-control-0.0.1-SNAPSHOT.jar
``` 

**В домашней директории должен появиться файл monitoring.mv.db**

## Общее описание сервиса

Сервис обеспечивает (интерфейс доступен по ссылке):
- сбор данных о работе дорожных контроллеров (длительность фаз в определенный момент времени, момент переключения каждой из фаз) в виде нечеловекочитаемого (зашифрованного) бинарного файла.
- возможность просмотреть данные о работе системы мониторинга дорожных контроллеров.
- возможность управления программами как отдельного дорожного контроллера, так и выбранной совокупности дорожных контроллеров и переключения программ в необходимый момент с учетом ограничений возможности переключения фаз и длины времени через удобный интерфейс. Реализованный алгоритм обеспечивает бесшовный переход от одной программы к другой в заданное время (время перехода от одной программы к другой можно настроить в интерфейсе).
- валидацию получаемых данных об изменении состояния дорожных контроллеров и отслеживания аномалий в их штатной работе. Алгоритм позволяет выявлять аномалии в работе дорожного контроллера и обращать внимание оператора, интеллектуально выявляя «инциденты» в работе дорожного контроллера
- визуальный поиск экстремумов по параметрам «скорость», «занятость», «загруженность» (будет реализовано позднее)

## Killer-фичи решения:

Использование бинарного файла в процессе сбора данных о работе дорожных контроллеров:
- возможность общего доступа к данным через понятный язык SQL,
- возможность введения ограничений для обеспечения безопасности,
- возможность обеспечения целостности данных, сокращение избыточности данных 

OpenAPI (доступен у разработанного сервиса): 
- позволяет реализовать интеграцию с различными дорожными контроллерами

## Использованные технологии:
- интерфейс реализован на Java
- in-memory база данных H2
